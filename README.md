## Get started

- Download [VSCode](https://code.visualstudio.com/docs/?dv=win)
- Download [git](https://git-scm.com/downloads)

## Installation
1. Clone this repository
```
git clone https://github.com/CatOnTheMonn/Cats
```
2. Run in terminal 
```
python3 main.py
```
3. Visit `http://127.0.0.1:5000`
